# Enterprise Microsoft 365 Administration Portfolio

## Overview

This repository documents a hands-on Microsoft 365 administration lab designed to simulate common enterprise IT administration and support responsibilities.

The portfolio was built using a Microsoft 365 Business Premium environment and covers tenant administration, user and group management, licensing, Exchange Online, Microsoft Teams, SharePoint Online, OneDrive, identity security, help desk operations, and end-to-end employee lifecycle administration.

The projects progress from individual Microsoft 365 administrative tasks to realistic IT support scenarios and a complete employee onboarding case study.

# User Lifecycle Administration

A major focus of this portfolio is understanding the Microsoft 365 employee lifecycle.

```text
JOINER
  │
  ├── Create identity
  ├── Assign license
  ├── Assign groups
  ├── Configure collaboration access
  └── Verify authentication
         │
         ▼
MOVER
  │
  ├── Modify group membership
  ├── Modify resource access
  ├── Adjust licenses
  └── Review permissions
         │
         ▼
LEAVER
  │
  ├── Block sign-in
  ├── Revoke sessions
  ├── Remove access
  ├── Review mailbox data
  ├── Review OneDrive data
  ├── Recover licenses
  └── Preserve required organizational data
```
