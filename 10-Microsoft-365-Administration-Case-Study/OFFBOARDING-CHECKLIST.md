# Microsoft 365 User Offboarding Checklist

- [ ] Confirm employee departure
- [ ] Block user sign-in
- [ ] Revoke active sessions
- [ ] Reset account credentials
- [ ] Remove group memberships
- [ ] Remove Teams memberships
- [ ] Review shared mailbox permissions
- [ ] Review Exchange mailbox requirements
- [ ] Review OneDrive organizational data
- [ ] Review SharePoint access
- [ ] Remove Microsoft 365 license
- [ ] Preserve required organizational data
- [ ] Document account closure
